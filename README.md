## :pushpin: 简介
>> 这是一个模仿博客园的博客平台，主要是为前端学习者作为练手素材，或则对于有前端经验想自行搭建设计博客页面的开发者使用
## :bulb: 背景
>> 本人以前在组内学习前端时，练手项目大多使用网上像“天气预报”等接口进行，但却遇到较多困难。比如跨域问题: 大多API都采用是json格式，但要想解决跨域问题最简单的却要用jsonp格式。本人当时找了好多方法，大多都要改后台代码，这对于前端开发者是很不友好的。于是在本人成为学长后，为避免学弟学妹遇到像我一样的问题，决定开发一友好交互的后台提供给学习前端的学弟学妹使用
## :label: 项目介绍
>> 本系统采用SpringMVC + Spring + Mybatis 进行开发
>> 使用Shiro进行权限管理
>> 使用solr进行搜索
>> 使用EasyUI编写后台主页面
>> 使用百度ueditor编辑博客
>> 前台页面又前端开发者自行设计[前端接口文档]()

开发流程：

	1:需求分析
	2:系统设计
	3:环境搭建
	4:数据库设计
	5:页面编写
	
	6：系统控制：刷新系统缓存
	7：博主信息管理功能实现:修改密码，修改博主基本信息，查找博主信息，注销账号
	8：博客管理功能实现：写博客，修改博客，查找所有博客（分页），又ID查找博客，删除博客
	9：博客类型管理实现：添加新博客类型，修改已有博客类型，查找所有博客类型（分页），删除博客类型
	10：评论管理：审核评论，查询评论（分页），删除评论
	11：友情链接管理：添加链接，修改链接，查询链接（分页），删除链接
	后台管理基本完成
